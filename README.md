<h1>Currency Converter Chrome Extension </h1>
A simple and clean Chrome extension that converts currency values using a live exchange-rate API.

This project is beginner-friendly and built using plain **HTML, CSS, and JavaScript**.

---

 Features

- Convert USD to multiple currencies
- Real-time exchange rates
- Clean and minimal UI
- Free API usage
- Easy to install (no payment required)

---

Tech Stack

- HTML
- CSS
- JavaScript
- Fetch API
- Chrome Extensions 

---

currency-extension/
├── manifest.json
├── index.html 
├── exchange.js

----

 How to Install (Free)

1. Open this GitHub repository
2. Click the green **Code** button
3. Select **Download ZIP**  
   (GitHub automatically creates the ZIP for you)
4. Extract the downloaded ZIP file
5. Open **Google Chrome**
6. Go to:
7. Enable **Developer Mode** (top-right)
8. Click **Load unpacked**
9. Select the extracted project folder

 The extension will now be installed.
